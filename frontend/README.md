# Frontend Developer Skills

AI agent skills untuk developer frontend. Setiap skill berisi rules, conventions, checklist, dan template yang membuat AI agent menghasilkan kode frontend produksi.

## Skills

| # | Skill | File |
|---|-------|------|
| 1 | React Component Builder | `react-component/SKILL.md` |
| 2 | Next.js App Router | `nextjs-app-router/SKILL.md` |
| 3 | Tailwind CSS Design System | `tailwind-design-system/SKILL.md` |
| 4 | TypeScript Strict Mode | `typescript-strict/SKILL.md` |
| 5 | Frontend Testing Suite | `frontend-testing/SKILL.md` |
| 6 | Web Performance & SEO | `web-performance-seo/SKILL.md` |

## Usage

Copy the skill folder(s) into your AI agent's skills directory, or reference the SKILL.md file directly in your agent configuration.
