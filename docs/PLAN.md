## Skill File Structure

Tiap skill adalah folder berisi:

```
skill-name/
├── SKILL.md          # Main instruction (dibaca AI agent)
├── rules/            # Aturan spesifik
│   ├── naming.md     # Naming conventions
│   ├── do.md         # Best practices (lakukan ini)
│   └── dont.md       # Anti-patterns (hindari ini)
├── templates/        # Template siap pakai
│   └── component.tsx
├── examples/         # Contoh good vs bad
│   ├── good/
│   └── bad/
└── checklist.md      # Review checklist sebelum commit
```
